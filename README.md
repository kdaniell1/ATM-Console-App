# ATM Console Application (Java)

A simple ATM simulation built with basic Java (no loops or OOP yet).
This is Version 1, built during Week 1 of learning Java.

## Features
- PIN authentication
- Check account balance
- Deposit money
- Withdraw money (with insufficient funds check)
- Clean and simple console interface

## Tools Used
- Java (JDK 17)
- IntelliJ IDEA
- Scanner class for user input

- Convert to OOP (classes & objects)
- Add loops to allow continuous transactions
- Add multiple users
